# Major Social & Communication Platforms Overview

---

## 1. Discord

**Discord** is a real-time voice, video, and text communication platform.

- **Founded:** 2015  
- **Founders:** Jason Citron, Stan Vishnevskiy  
- **Headquarters:** United States  
- **Primary Use:** Communities, gaming, study groups, tech servers  

### Key Features
- Servers (community spaces)
- Channels (text & voice)
- Roles & permissions
- Bots & automation
- Screen sharing
- Stage channels
- Nitro subscription (premium features)

### Common Uses
- Gaming communities  
- Developer communities  
- Study groups  
- Private friend groups  

---

## 2. YouTube

**YouTube** is a video-sharing platform owned by Google.

- **Founded:** 2005  
- **Founders:** Steve Chen, Chad Hurley, Jawed Karim  
- **Owner:** Google (Alphabet Inc.)  

### Key Features
- Video uploads
- Live streaming
- Shorts (short-form video)
- Monetization via ads
- Community posts

### Common Uses
- Entertainment
- Education
- Tutorials
- Music streaming
- Content creation business

---

## 3. Facebook

**Facebook** is a social networking platform.

- **Founded:** 2004  
- **Founder:** Mark Zuckerberg  
- **Owner:** Meta Platforms  

### Key Features
- Profiles & pages
- Groups
- Marketplace
- Messenger
- Ads platform

### Common Uses
- Social networking
- Business marketing
- Community groups
- Local buying/selling

---

## 4. X (formerly Twitter)

**X** is a microblogging platform.

- **Founded:** 2006  
- **Original Name:** Twitter  
- **Owner:** Elon Musk (since 2022)  

### Key Features
- Short posts ("tweets")
- Trending topics
- Spaces (live audio)
- Subscriptions
- Real-time news

### Common Uses
- News updates
- Public discussions
- Influencer content
- Tech & politics debates

---

## 5. Reddit

**Reddit** is a community-based discussion platform.

- **Founded:** 2005  
- **Founders:** Steve Huffman, Alexis Ohanian  

### Key Features
- Subreddits (topic-based communities)
- Upvotes & downvotes
- AMAs (Ask Me Anything)
- Anonymous posting

### Common Uses
- Tech discussions
- Memes
- Q&A
- News aggregation

---

## 6. Dread

**Dread** is a discussion platform similar to Reddit, mainly accessible via Tor.

- **Launched:** 2018  
- **Access:** Dark web (.onion)  

### Key Features
- Reddit-style interface
- Privacy-focused
- Anonymous access

### Common Uses
- Privacy discussions
- Underground communities
- Dark web forums

---

## 7. Telegram

**Telegram** is a cloud-based messaging app.

- **Founded:** 2013  
- **Founder:** Pavel Durov  

### Key Features
- Channels (broadcast)
- Groups (up to 200k+ members)
- Bots
- End-to-end encryption (Secret Chats)
- File sharing (large files)

### Common Uses
- Messaging
- Community broadcasting
- Crypto communities
- File distribution

---

## 8. Endchan

**Endchan** is an imageboard platform.

- **Launched:** 2015  
- **Type:** Anonymous imageboard  

### Key Features
- Anonymous posting
- Topic boards
- Minimal moderation (varies by board)

### Common Uses
- Free speech discussions
- Anonymous forums

---

## 9. WhatsApp

**WhatsApp** is a messaging platform owned by Meta.

- **Founded:** 2009  
- **Founders:** Brian Acton, Jan Koum  
- **Owner:** Meta Platforms  

### Key Features
- End-to-end encryption
- Voice & video calls
- Status updates
- Group chats
- Business accounts

### Common Uses
- Personal messaging
- Business communication
- Family groups

---

## 10. LinkedIn

**LinkedIn** is a professional networking platform.

- **Founded:** 2003  
- **Owner:** Microsoft  

### Key Features
- Professional profiles
- Job listings
- Networking
- Skill endorsements
- Business content sharing

### Common Uses
- Job searching
- Professional branding
- Corporate networking
- Recruiting

---

## 11. Bluesky (bsky)

**Bluesky** is a decentralized social media platform.

- **Founded:** 2019 (as a Twitter project)  
- **Public Launch:** 2023  
- **Protocol:** AT Protocol  

### Key Features
- Decentralized identity
- Custom feeds
- Moderation tools
- Microblogging style posts

### Common Uses
- Alternative to X
- Open social networking
- Decentralized communities
