# ESP8266 Documentation

## 1. Origin of ESP8266

The **ESP8266** is a low-cost Wi-Fi microcontroller developed by **Espressif Systems** (Shanghai, China).

- **Company:** Espressif Systems  
- **Initial Release:** 2014  
- **Purpose:** Affordable Wi-Fi connectivity for embedded systems  
- **Architecture:** Tensilica Xtensa LX106  

The ESP8266 became extremely popular due to its low price and built-in Wi-Fi, revolutionizing IoT development for hobbyists and startups.

---

## 2. Architecture Overview

The ESP8266 integrates:

- 32-bit microcontroller (Xtensa LX106)
- Built-in 2.4 GHz Wi-Fi (802.11 b/g/n)
- TCP/IP stack
- SRAM
- SPI flash interface
- GPIO and common peripherals

### CPU Specifications

- Single-core 32-bit
- Clock Speed: 80 MHz (default), 160 MHz (overclock)
- Harvard architecture
- Integrated Wi-Fi MAC/PHY

---

## 3. ESP8266 Variants (Modules)

ESP8266 comes in multiple module versions manufactured by various vendors:

### 3.1 ESP-01
- Minimal GPIO (2 usable pins)
- Small size
- 512KB–1MB flash (varies)

### 3.2 ESP-07
- More GPIO access
- External antenna option

### 3.3 ESP-12E / ESP-12F
- Most popular module
- Full GPIO access
- PCB antenna
- 4MB flash (common)

### 3.4 NodeMCU (Development Board)
- Based on ESP-12 module
- USB-to-Serial onboard
- Easy programming via USB
- Breadboard friendly

### 3.5 Wemos D1 Mini
- Compact development board
- USB interface
- Shield ecosystem

---

## 4. Key Features

### Wireless Connectivity
- 2.4 GHz Wi-Fi
- 802.11 b/g/n support
- Station (STA) mode
- Access Point (AP) mode
- Station + AP mode
- TCP/IP stack built-in

### Processing
- 80 MHz default
- 160 MHz optional
- Single-core MCU

### Memory
- ~50KB usable RAM (dynamic)
- External SPI Flash (512KB to 16MB depending on module)

### GPIO & Peripherals
- Up to 17 GPIOs (depends on module)
- SPI
- I2C (software-based)
- I2S
- UART (2, one TX-only)
- PWM (software-based)

### Analog Features
- 10-bit ADC
- 0–1V input range (raw chip)
- Many dev boards scale to 0–3.3V

---

## 5. Power Modes

- Active mode
- Modem sleep
- Light sleep
- Deep sleep

Deep sleep current can go down to ~20 µA.

## 6. Programming & Development

ESP8266 can be programmed using:

- Arduino IDE (most popular)
- ESP8266 Non-OS SDK
- ESP8266 RTOS SDK
- PlatformIO
- MicroPython
- Lua (NodeMCU firmware)

### Supported Languages

- C
- C++
- Lua
- MicroPython

## 8. Comparison with ESP32

| Feature | ESP8266 | ESP32 |
|----------|----------|--------|
| CPU | Single-core | Single/Dual-core |
| Max Clock | 160 MHz | 240 MHz |
| Bluetooth | No | Yes |
| ADC | 10-bit | 12-bit |
| GPIO | Limited | More |
| Security | Basic | Advanced |
| Cost | Very low | Low |

---

## 9. Advantages

- Extremely cheap
- Built-in Wi-Fi
- Large community support
- Easy Arduino integration
- Small form factor

---

## 10. Limitations

- No Bluetooth
- Limited RAM
- Single ADC channel
- Software-based PWM
- Can become unstable under heavy memory usage

---

## 11. Historical Impact

The ESP8266 changed the IoT market by making Wi-Fi-enabled microcontrollers affordable for everyone. It paved the way for the more advanced ESP32 and remains widely used in low-cost embedded systems.

---

## 12. Conclusion

The ESP8266 remains one of the most influential microcontrollers in IoT history. While newer chips like the ESP32 offer more features, the ESP8266 is still an excellent choice for simple, low-cost Wi-Fi projects.
