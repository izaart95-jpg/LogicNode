# Platforms, Communities & Cybersecurity Intelligence Sources

This document covers social platforms, cybersecurity news outlets, threat intelligence organizations, underground forums, and research communities in one unified reference.

---

# Part 1: Social & Communication Platforms

## 1. Discord

**Discord** is a real-time voice, video, and text communication platform.

- **Founded:** 2015
- **Founders:** Jason Citron, Stan Vishnevskiy
- **Primary Use:** Communities, gaming, study groups, tech servers

### Key Features
- Servers (community spaces) with channels (text & voice)
- Roles & permissions system
- Bots & automation
- Screen sharing & Stage channels
- Nitro subscription (premium features)

### Cybersecurity Context
- Used by cybersecurity communities, bug bounty groups, OSINT teams
- Also abused by threat actors for C2 and malware distribution

---

## 2. YouTube

**YouTube** is a video-sharing platform owned by Google (Alphabet Inc.).

- **Founded:** 2005
- **Founders:** Steve Chen, Chad Hurley, Jawed Karim

### Key Features
- Video uploads, live streaming, Shorts
- Monetization via ads
- Community posts

---

## 3. Facebook

**Facebook** is a social networking platform owned by Meta Platforms.

- **Founded:** 2004
- **Founder:** Mark Zuckerberg

### Key Features
- Profiles, pages, groups, Marketplace
- Messenger, Ads platform

---

## 4. X (formerly Twitter)

**X** is a microblogging platform.

- **Founded:** 2006
- **Owner:** Elon Musk (since 2022)

### Key Features
- Short posts, trending topics, Spaces (live audio)
- Real-time news and public discussions

---

## 5. Reddit

**Reddit** is a community-based discussion platform.

- **Founded:** 2005
- **Founders:** Steve Huffman, Alexis Ohanian

### Key Features
- Subreddits (topic-based communities)
- Upvotes & downvotes, AMAs

### Cybersecurity Subreddits
- r/netsec
- r/cybersecurity
- r/ReverseEngineering

---

## 6. Telegram

**Telegram** is a cloud-based messaging app.

- **Founded:** 2013
- **Founder:** Pavel Durov

### Key Features
- Channels (broadcast), Groups (200k+ members)
- Bots, end-to-end encryption (Secret Chats)
- Large file sharing

### Cybersecurity Context
- Widely used by threat actors, leak groups, ransomware affiliates
- Also used by OSINT researchers and crypto communities

---

## 7. WhatsApp

**WhatsApp** is a messaging platform owned by Meta.

- **Founded:** 2009
- **Founders:** Brian Acton, Jan Koum

### Key Features
- End-to-end encryption, voice & video calls
- Group chats, Business accounts

---

## 8. LinkedIn

**LinkedIn** is a professional networking platform owned by Microsoft.

- **Founded:** 2003

### Key Features
- Professional profiles, job listings
- Skill endorsements, business content sharing

---

## 9. Bluesky (bsky)

**Bluesky** is a decentralized social media platform.

- **Founded:** 2019 (as a Twitter project)
- **Protocol:** AT Protocol

### Key Features
- Decentralized identity, custom feeds
- Microblogging style, moderation tools

---

## 10. Dread

**Dread** is a Reddit-style discussion platform accessible via Tor.

- **Launched:** 2018
- **Access:** Dark web (.onion)

### Key Features
- Privacy-focused, anonymous access
- Underground and privacy-focused discussions

---

## 11. Endchan

**Endchan** is an anonymous imageboard platform.

- **Launched:** 2015

### Key Features
- Anonymous posting, topic boards
- Minimal moderation

# Part 7: APT & Threat Reports

APT reports typically include:
- Attribution analysis
- TTP mapping (MITRE ATT&CK)
- Indicators of Compromise (IOCs)
- Malware reverse engineering
- Infrastructure tracking

**Key publishers:** Kaspersky (Securelist), Insikt Group, Citizen Lab, Flashpoint, DarkOwl, Intel 471
